From Hero to Zero

Being the "heroic" systems person is not something to strive for in the quest for mastery in the Operations discipline. In this talk, Jennifer will describe her journey to take a complex project deployment and configuration that requires a lot of manual intervention to a state of zero heroics. This talk will be tool agnostic focusing on the process of discovery as there is no implementation that is a one size fit all solution. The key takeaway from this talk will be to help practitioners see how to create a plan towards building a more reliable, consistent, and high-quality service.

About Jennifer Davis

Jennifer (@sigje) has worked in education, startup, and large-scale environments, which has contributed to a diverse set of experiences. Currently she is a lead senior production engineer at Yahoo helping to transform Operations work.

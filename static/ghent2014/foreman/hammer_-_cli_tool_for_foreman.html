Hammer - CLI tool for Foreman

3 February 16:40 - 17:00 Tomas Strachota

The Foreman team recently started working on an officially supported cli tool called Hammer. The goal of the project is to create a full featured command line client that reflects UI functionality of the Foreman. It aims to allow for effectiveness when being used interactively as well as easy scripting. In this talk I'd like to show it's main features, demonstrate couple of common usecases and present technologies used behind.

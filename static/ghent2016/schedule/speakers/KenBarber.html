#cfgmgmtcamp

1 and 2 February 2016

Gent, Belgium

Puppet Query Language

Ken Barber

Puppet - Monday, 1/2/2016 16:20 - 17:00- B1.015

This session will be all about the new Puppet query language: PQL. This new language is a more friendly query language compared to the existing PuppetDB language, designed to be more user-facing, while still keeping the power. Here we’ll talk about the background, walk through some of the features and show examples of how it is used. We’ll also talk about the future development possibilities for this feature.

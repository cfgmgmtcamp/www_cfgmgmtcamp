#cfgmgmtcamp

6 and 7 February 2017

Gent, Belgium

Viewing potential Puppet catalog changes from the Foreman UI

Greg Sutcliffe - Monday, Feb 6, 15:00 - 15:20 - B2.015

Octocatalog-diff is a commandline tool from GitHub to show the differences between two puppet environments for a specific node *before* the node is moved to the new env. That's great, but not everyone wants to work on the commandline

In this talk we'll look at a new Foreman plugin (foreman-puppetdiff) which brings this functionality to the Foreman UI. We'll cover the basics of octocatalog-diff (installation, uses, limitations) and then show how to set up the Foreman plugin and use it in the UI.

Foreman Community Update 2016/17

Greg Sutcliffe - Tuesday, Feb 7, 14:00 - 14:20 - B2.015

In this short session, we'll recap what's been achieved in the last year since CfgMgmt 2016 (with the help of spiffy metrics!) and together brainstorm some of the challenges that we might want to tackle in 2017.

This talk is for you if you're interested in learning about new features that you might have missed, or want to put forward your own ideas for things to discuss.

About Speaker

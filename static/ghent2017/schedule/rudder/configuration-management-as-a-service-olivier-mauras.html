#cfgmgmtcamp

6 and 7 February 2017

Gent, Belgium

Rudder and configuration management as a service - #cfgmgmtaas

Olivier Mauras - Monday, February 6, 15:00 - 15:20 - B4.042

In a devops team there's a good chance that everyone is using the same tools. From OS configuration to app deployment everyone is aligned.

In an IT operations context, where system, application and development teams are different, it can easily become a burden to align the tools especially when talking about configuration management and application deployment.

What if ops and devs teams could share the same tools without messing with each others?
What if we could have configuration management as a service usable by different teams in total autonomy?

Let's see how this can be done with Rudder.

About Olivier Mauras

Olivier Mauras is a Linux system engineer specialized in automation and configuration management.
